# arXiv Search Tool

A simple command-line tool to search the arXiv API and save results as human-readable JSON files.

## Setup

1. **Create a virtual environment** (recommended):
   ```bash
   python3 -m venv venv
   ```

2. **Activate the virtual environment**:
   ```bash
   source venv/bin/activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Add your OpenAI API key** (needed for plain-English summaries):

   Edit the `.env` file and replace the placeholder with your key:
   ```
   OPENAI_API_KEY=your_key_here
   ```

5. **Configure the workspace** (required before first use):
   ```bash
   python configure.py
   ```
   Normalizes file names for consistent path resolution across platforms. The search script will not run without this step.

## Quick Start

Run a search:
```bash
./venv/bin/python arxiv_search.py --search-query 'all:machine learning'
```

Results are automatically saved to a timestamped JSON file (e.g., `arxiv_search_2026-01-21_15-04-51.json`).

## Examples

Search for recent papers on reinforcement learning from human feedback:

```bash
python arxiv_search.py \
  --search-query 'all:"reinforcement learning" AND all:"human feedback"' \
  --sort-by submittedDate \
  --sort-order descending \
  --max-results 20
```

Search for transformer architecture papers in the ML category:

```bash
python arxiv_search.py \
  --search-query 'ti:transformer AND cat:cs.LG' \
  --sort-by relevance \
  --max-results 30
```

Fetch specific papers by arXiv ID:

```bash
python arxiv_search.py --id-list '1706.03762,2005.14165'
```

Run a search without GPT summaries (faster, no API key needed):

```bash
python arxiv_search.py \
  --search-query 'abs:"diffusion model"' \
  --no-summarize
```

## Search Query Syntax

The arXiv API uses a specific query syntax:

| Prefix | Searches |
|--------|----------|
| `all:` | All fields (title, abstract, authors, etc.) |
| `ti:` | Title only |
| `au:` | Author name |
| `abs:` | Abstract only |
| `cat:` | Category (e.g., `cs.LG`, `cs.AI`, `physics.optics`) |

### Boolean Operators

- `AND` ‚Äî both terms must match
- `OR` ‚Äî either term matches
- `ANDNOT` ‚Äî exclude terms
- Use parentheses `()` to group terms

### Examples

| Query | Description |
|-------|-------------|
| `all:transformer` | Papers mentioning "transformer" anywhere |
| `ti:attention` | Papers with "attention" in the title |
| `au:hinton` | Papers by author Hinton |
| `cat:cs.LG` | Machine Learning category papers |
| `all:"neural network"` | Exact phrase match |
| `all:GPT AND all:reasoning` | Papers about GPT and reasoning |
| `cat:cs.AI OR cat:cs.LG` | AI or ML category papers |

## Command-Line Options

| Option | Default | Description |
|--------|---------|-------------|
| `--search-query` | (required) | arXiv search query string |
| `--id-list` | ‚Äî | Fetch specific arXiv IDs (comma-separated) |
| `--max-results` | 50 | Total number of results to retrieve |
| `--sort-by` | ‚Äî | Sort by: `relevance`, `lastUpdatedDate`, or `submittedDate` |
| `--sort-order` | ‚Äî | Sort order: `ascending` or `descending` |
| `--output` | ‚Äî | Custom output file path |
| `--output-dir` | `.` | Directory for auto-generated output files |
| `--no-summarize` | ‚Äî | Skip GPT-powered plain-English summaries |
| `--stdout` | ‚Äî | Print raw JSONL to terminal instead of saving |
| `--start` | 0 | Starting index for pagination |
| `--chunk-size` | 200 | Results per API request |
| `--delay-seconds` | 3.0 | Delay between paginated requests |

## Output Format

Results are saved as JSON with this structure:

```json
{
  "search_metadata": {
    "timestamp": "2026-01-21T15:04:51.281766",
    "search_query": "your search query",
    "total_results": 20
  },
  "results": [
    {
      "title": "Paper Title",
      "published": "2026-01-20T14:53:32Z",
      "url": "http://arxiv.org/abs/2601.14032v1",
      "abstract": [
        "First line of the abstract wrapped at 80 characters...",
        "Second line continues here...",
        "And so on for readability."
      ],
      "plain_english_summary": [
        "A short, jargon-free explanation of what the paper is",
        "about, generated by GPT-5.2."
      ]
    }
  ]
}
```

The `plain_english_summary` field is omitted when running with `--no-summarize`.

## Tips

- **Be polite to the API**: The default 3-second delay between requests is recommended by arXiv
- **Date filtering**: arXiv API doesn't support date ranges in queries. Sort by `submittedDate` descending to get the newest papers first
- **Large searches**: Maximum 30,000 results total, 2,000 per request
